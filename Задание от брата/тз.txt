- паттерн MVC (Spring MVC - реализация его)


-------

Реализовать приложение для учёта книг (прочитанные, запланированные, отложенные и т.п.).
Данные должны сохраняться в базу данных или Excel (CSV) табличку.

1. Консольное приложение.
 1.1 Стартовый экран (меню)
    - 1. Добавить книгу (переход на экран 1.2)
    - 2. Найти книгу
    - 3. Найти автора
    - 4. Показать список книг
    - 5. Показать статистику
    - 6. Выход
 2.1 Экран добавления книги
    * Название книги: (пользовательский ввод)
    * Автор: (пользовательский ввод)
    * 0 - в список для чтения / 1 - прочитана / 2 - прослушана
    - (если в предыдущем пунке выбран п. 1 или 2) Дата прочтения: (форматы: ДД.ММ.ГГГГ, ММ.ГГГГ, ГГГГ, приблизительная дата с знаком ~)
 3.1 Поиск книги по названию (ввод - название, вывод - название, автор, статус (прочитана, прослушана, для чтения), дата прочтения (если есть))
